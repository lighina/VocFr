## 📚 学习系统

### Unite 结构

VocFr 采用三级学习结构：**Unite → Section → Words**

```
Unite 1: À l'école (在学校)
├── Section 1: Dans la classe (教室里)
│   ├── livre (书)
│   ├── stylo (笔)
│   ├── cahier (笔记本)
│   └── ... (10-15 个单词)
├── Section 2: Les personnes (人物)
│   └── ...
└── Section 3: Les actions (动作)
    └── ...
```

### 6 个主题 Unite

| Unite | 主题 | 词汇量 | 解锁要求 |
|-------|------|--------|----------|
| Unite 1 | À l'école (在学校) | ~60 词 | 初始解锁 |
| Unite 2 | Les couleurs (颜色) | ~45 词 | 30 ⭐ |
| Unite 3 | Les animaux (动物) | ~60 词 | 45 ⭐ |
| Unite 4 | À la maison (在家里) | ~60 词 | 60 ⭐ |
| Unite 5 | La nourriture (食物) | ~60 词 | 75 ⭐ |
| Unite 6 | Le corps (身体) | ~45 词 | 90 ⭐ |

**总计**: 约 330+ 核心法语词汇

### 单词信息

每个单词卡片包含：

**正面**：
- 法语原形 (canonical form)
- 音频播放按钮
- 词性标注 (noun, verb, adjective...)
- 性别标注 (masculine/feminine，针对名词)

**背面**：
- 中文释义
- 词汇分类 (category)
- 省音标记 (如适用)

**示例**：
```
正面: livre 🔊 | noun (m)
背面: 书
分类: school_objects
```

## 🎓 Study Mode（学习模式）

### 功能特点

**系统化学习**：
- 按主题分类学习
- 循序渐进的难度设计
- 完整的词汇覆盖

**交互式卡片**：
- 点击翻转查看释义
- 音频播放学习发音
- 滑动切换下一张卡片

**学习追踪**：
- 标记已学习的 Section
- 自动保存学习进度
- 可随时返回复习

### 学习技巧

**首次学习**：
1. 先听音频，模仿发音
2. 点击卡片查看中文释义
3. 尝试记忆单词和释义
4. 浏览完所有单词后标记已学习

**复习学习**：
1. 再次进入已学习的 Section
2. 测试自己能否记住释义
3. 重点复习不熟悉的单词
4. 利用音频强化记忆

## ✅ Test Mode（测试模式）

### 测试规则

**测试范围**：
- 整个 Unite 的所有单词
- 随机抽取题目
- 每次测试题目可能不同

**题目类型**：
- 单选题（4 个选项）
- 法语 → 中文
- 即时反馈对错

**评分系统**：

| 正确率 | 星星 | 等级 |
|--------|------|------|
| ≥ 90% | ⭐⭐⭐ | 优秀 |
| ≥ 70% | ⭐⭐ | 良好 |
| ≥ 60% | ⭐ | 及格 |
| < 60% | 无 | 需复习 |

### 测试策略

**准备阶段**：
- 确保学完所有 Section
- 至少复习 1-2 遍
- 玩游戏模式热身

**测试中**：
- 仔细阅读题目
- 注意词性和性别
- 不确定时用排除法

**测试后**：
- 查看错误题目
- 针对性复习
- 重新测试刷高分

### 星星系统

**用途**：
- 解锁新的 Unite
- 显示学习成就

**累积规则**：
- 星星永久保留
- 可重复测试获得更多星星
- 每次测试都计入总星星数

**解锁要求**：
```
Unite 1: 0 ⭐  (初始解锁)
Unite 2: 30 ⭐
Unite 3: 45 ⭐
Unite 4: 60 ⭐
Unite 5: 75 ⭐
Unite 6: 90 ⭐
```

## 🎮 Games Mode（游戏模式）

### 猜词游戏

**游戏玩法**：
1. 看到中文提示
2. 从字母中拼出法语单词
3. 点击字母按钮组成单词
4. 正确拼写得分

**提示信息**：
- 单词长度（下划线数量）
- 词性和性别
- 词汇分类

**计分规则**：
- 正确拼写：+10 分
- 连续正确：额外奖励
- 使用提示：减少分数
- 完成游戏：获得宝石

**难度选择**：
- 可选择不同 Unite
- 只能玩已学习的 Unite
- 后面的 Unite 更有挑战

### 未来游戏（计划中）

**记忆配对**：
- 翻转卡片找配对
- 法语 - 中文匹配
- 计时挑战

**拼写比赛**：
- 听音频拼写
- 限时答题
- 排行榜竞争

## 📖 Storybook Mode（故事书模式）

### 故事书结构

每个 Unite 配有 1-3 本主题故事：

**Unite 1 故事书**：
1. 📗 Le Premier Jour d'École (第一天上学) - 免费
2. 📘 Les couleurs de ma classe (我的教室的颜色) - 10 💎
3. 📙 Le stylo magique de Léo (Léo 的魔法笔) - 10 💎

**故事类型**：
- **双语故事**：法语 + 中文翻译
- **纯法语故事**：仅法语（进阶）

### 解锁规则

**第一本故事书**：
- 完成该 Unite 测试
- 正确率 ≥ 60%
- 自动解锁，无需宝石

**其他故事书**：
- 需要 10 💎 宝石
- 通过游戏模式获得宝石
- 解锁后永久可读

### 阅读功能

**基本操作**：
- **翻页**：左右滑动或点击屏幕边缘
- **自动播放**：翻页时自动朗读该页
- **进度保存**：自动记录阅读位置

**显示模式**：

**iPhone / iPad 竖屏**：
```
┌─────────────────┐
│                 │
│  [  单页显示  ] │
│                 │
│  法语文本        │
│  中文翻译        │
│                 │
└─────────────────┘
```

**iPad 横屏**：
```
┌────────────────────────────────┐
│  [左页]  ┃书脊┃  [右页]       │
│                                │
│  法语文本  ┃  ┃  法语文本      │
│  中文翻译  ┃  ┃  中文翻译      │
│                                │
└────────────────────────────────┘
```

## 💎 奖励系统详解

### 星星系统 (Stars)

**获取方式**：

| 活动 | 星星数 |
|------|--------|
| 测试 60%-69% | ⭐ (1 星) |
| 测试 70%-89% | ⭐⭐ (2 星) |
| 测试 90%+ | ⭐⭐⭐ (3 星) |

**使用方式**：
- 解锁新 Unite
- 不消耗，永久保留
- 显示学习成就

### 宝石系统 (Gems)

**获取方式**：

| 活动 | 宝石数 |
|------|--------|
| 完成猜词游戏 | 5-10 💎 |
| 达成成就 | 10-50 💎 |
| 连续学习奖励 | 5 💎 |

**使用方式**：
- 解锁故事书（10 💎/本）
- 未来功能：主题、头像等
- 消耗型，使用后扣除

### 成就系统（计划中）

**学习成就**：
- 🏆 初学者：完成 Unite 1
- 🏆 进阶者：完成 Unite 1-3
- 🏆 精通者：完成全部 Unite

**测试成就**：
- 🏆 完美主义者：某 Unite 达到 100%
- 🏆 全优生：所有 Unite ≥ 90%

**游戏成就**：
- 🏆 拼字高手：游戏正确率 95%+
- 🏆 连胜王：连续答对 20 题
