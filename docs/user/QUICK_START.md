## 📱 第一步：认识主界面

打开 VocFr 后，您会看到 4 个主要模块：

```
┌─────────────────────────────────┐
│                                 │
│    ⭐ 0 Stars    💎 0 Gems     │
│                                 │
│  ┌────────┐  ┌────────┐        │
│  │ Study  │  │  Test  │        │
│  │   📚   │  │   ✅   │        │
│  └────────┘  └────────┘        │
│                                 │
│  ┌────────┐  ┌────────┐        │
│  │ Games  │  │Storybook│       │
│  │   🎮   │  │   📖   │        │
│  └────────┘  └────────┘        │
│                                 │
└─────────────────────────────────┘
```

**界面说明**：
- **顶部**：显示您的星星和宝石数量
- **Study**：系统学习新单词
- **Test**：测试赚取星星
- **Games**：游戏巩固记忆
- **Storybook**：阅读法语故事

---

## 🚀 第二步：开始学习

### 1. 进入学习模式

点击 **Study** 按钮 → 选择 **Unite 1: À l'école**

> 💡 **提示**：第一个 Unite 默认已解锁，可以直接开始学习

### 2. 选择第一个 Section

点击 **Section 1: Dans la classe** (在教室)

### 3. 学习单词

您会看到单词卡片：

```
┌──────────────────────┐
│                      │
│      livre          │
│                      │
│   🔊  |  noun (m)   │
│                      │
└──────────────────────┘
```

**操作方法**：
- **点击卡片**：翻转查看中文释义"书"
- **点击 🔊**：听法语发音
- **左右滑动**：切换下一个/上一个单词

### 4. 完成学习

浏览完所有单词后，点击 **标记为已学习**

---

## ✅ 第三步：测试检验

### 1. 进入测试模式

返回主界面 → 点击 **Test** → 选择 **Unite 1**

### 2. 开始答题

```
┌──────────────────────────────┐
│  Question 1/10               │
│                              │
│       livre                  │
│                              │
│  A. 书      B. 笔            │
│  C. 桌子    D. 椅子          │
│                              │
└──────────────────────────────┘
```

从 4 个选项中选择正确答案

### 3. 查看结果

完成后显示：

```
┌──────────────────────┐
│   🎉 测试完成！      │
│                      │
│   正确率: 85%        │
│   获得: ⭐⭐         │
│                      │
└──────────────────────┘
```

**评分标准**：
- ≥ 90%: ⭐⭐⭐
- ≥ 70%: ⭐⭐
- ≥ 60%: ⭐
- < 60%: 无星星

---

## 🎮 第四步：游戏练习

### 1. 进入游戏模式

主界面 → **Games** → **猜词游戏**

### 2. 拼写单词

看到中文提示"书"，从字母中拼出"livre"

```
┌──────────────────────────┐
│      中文: 书            │
│                          │
│    提示: noun (m)        │
│                          │
│    [  _  _  _  _  _  ]   │
│                          │
│  L  I  V  R  E  A  B  C │
│                          │
└──────────────────────────┘
```

点击字母按钮组成单词

### 3. 赚取宝石

每答对一题获得积分，完成后获得 💎 宝石

---

## 📖 第五步：阅读故事

### 1. 解锁故事书

完成 Unite 1 测试（≥60%）后，第一本故事书自动解锁

### 2. 开始阅读

主界面 → **Storybook** → 选择故事

```
┌────────────────────────────┐
│                            │
│   Le Premier Jour d'École  │
│                            │
│   [  封面图片  ]           │
│                            │
│   第一天上学                │
│   📖 开始阅读              │
│                            │
└────────────────────────────┘
```

### 3. 翻页阅读

- **滑动翻页**：左右滑动屏幕
- **点击翻页**：点击屏幕左右边缘
- **自动播放**：翻页时自动朗读该页内容

## ❓ 常见疑问

### Q: 第一次使用从哪里开始？

**A**: 点击 **Study** → 选择 **Unite 1** → 开始学习第一个 Section

### Q: 如何获得星星？

**A**: 完成测试可以获得星星。正确率越高，星星越多。

### Q: 星星有什么用？

**A**: 用星星解锁新的 Unite。例如 Unite 2 需要 30 颗星星。

### Q: 如何获得宝石？

**A**: 玩游戏可以获得宝石。宝石用于解锁更多故事书。

### Q: 可以跳过 Unite 直接学习吗？

**A**: 不建议！后面的 Unite 需要前面的基础。而且后面的 Unite 是锁定的，需要星星解锁。

### Q: 学习进度会保存吗？

**A**: 会！所有进度自动保存，随时可以继续学习。
