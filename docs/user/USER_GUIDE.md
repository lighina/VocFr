# VocFr 用户使用手册

欢迎使用 VocFr - 您的法语学习伙伴！

---

## 📱 关于 VocFr

VocFr 是一款基于游戏化学习理念的法语学习应用，通过多种学习模式帮助您轻松掌握法语词汇。应用采用日式极简美学设计，为您提供优雅舒适的学习体验。

### 主要特色

- 🎓 **系统化学习**：6 个主题单元，从基础到进阶
- 🎮 **游戏化学习**：通过游戏巩固记忆
- 📚 **互动故事书**：沉浸式法语阅读体验
- 💎 **奖励系统**：完成任务赚取宝石和星星
- 🌍 **多语言支持**：7 种界面语言

---

## 🚀 快速开始

### 首次使用

1. **启动应用**
   - 打开 VocFr 应用
   - 您将看到主界面，包含四个学习模块

2. **选择学习模式**
   - 点击 "Study" 开始系统学习
   - 从 Unite 1 开始您的法语学习之旅

3. **完成第一个 Section**
   - 学习 10-15 个单词
   - 点击单词卡片查看详细信息
   - 点击音频图标收听发音

4. **解锁更多内容**
   - 通过学习和测试赚取星星
   - 使用星星解锁新的单元

---

## 📚 学习模式详解

### Study（学习模式）

**功能**：系统化学习法语词汇

**使用方法**：

1. **选择 Unite**
   - 界面显示 6 个主题单元：
     - Unite 1: À l'école (在学校)
     - Unite 2: Les couleurs (颜色)
     - Unite 3: Les animaux (动物)
     - Unite 4: À la maison (在家里)
     - Unite 5: La nourriture (食物)
     - Unite 6: Le corps (身体)

2. **学习 Section**
   - 每个 Unite 包含多个 Section
   - 每个 Section 约 10-15 个单词
   - 按顺序学习效果最佳

3. **单词卡片功能**
   - **点击卡片**：翻转查看中文释义
   - **音频播放**：点击喇叭图标听发音
   - **词性显示**：名词、动词、形容词等
   - **性别标注**：阳性/阴性（针对名词）

4. **学习进度**
   - 完成学习后标记为已学习
   - 可随时返回复习

### Test（测试模式）

**功能**：检验学习成果，赚取星星

**使用方法**：

1. **选择 Unite 测试**
   - 只能测试已学习的 Unite
   - 测试包含该 Unite 所有单词

2. **答题流程**
   - 看到法语单词，选择正确的中文释义
   - 从 4 个选项中选择答案
   - 立即显示对错

3. **评分系统**
   - 测试结束显示正确率
   - 根据正确率获得星星：
     - ≥ 90%: 3 颗星 ⭐⭐⭐
     - ≥ 70%: 2 颗星 ⭐⭐
     - ≥ 60%: 1 颗星 ⭐
     - < 60%: 0 颗星

4. **星星用途**
   - 用于解锁新的 Unite
   - 每个 Unite 需要一定数量的星星才能解锁

### Games（游戏模式）

**功能**：通过游戏巩固词汇记忆

**猜词游戏**：

1. **游戏规则**
   - 看到单词的中文释义
   - 从字母中拼出正确的法语单词
   - 点击字母按钮组成单词

2. **提示系统**
   - 显示单词长度
   - 显示词性和性别
   - 可查看正确答案（但不计分）

3. **计分**
   - 正确拼写得分
   - 连续正确有额外奖励
   - 完成游戏获得宝石

**记忆卡片**（计划中）：
- 翻转卡片记忆单词
- 匹配游戏挑战

### Storybook（故事书模式）

**功能**：通过法语故事提升阅读能力

**使用方法**：

1. **解锁故事书**
   - 每个 Unite 有 1-3 本故事书
   - 第一本：完成测试（≥60%）自动解锁
   - 第二、三本：使用 10 宝石解锁

2. **阅读故事**
   - 翻页：左右滑动或点击屏幕边缘
   - 自动播放：翻页时自动朗读该页内容
   - 双语对照：法语原文 + 中文翻译（部分故事）

3. **iPad 特别功能**
   - 横屏时显示双页布局
   - 真实书籍阅读体验
   - 中间书脊效果

4. **阅读进度**
   - 自动记录阅读进度
   - 下次打开继续上次位置

## 💎 奖励系统

### 星星系统

**获取方式**：
- 完成 Unite 测试
- 正确率越高，获得星星越多

**用途**：
- 解锁新的 Unite
- 每个 Unite 需要特定数量的星星

**星星要求**：
- Unite 1: 0 颗星（初始解锁）
- Unite 2: 30 颗星
- Unite 3: 45 颗星
- Unite 4: 60 颗星
- Unite 5: 75 颗星
- Unite 6: 90 颗星

### 宝石系统

**获取方式**：
- 完成游戏挑战
- 达成特定成就
- 完成学习里程碑

**用途**：
- 解锁故事书（每本 10 宝石）
- 未来功能：主题、头像等

## 💡 学习建议

### 最佳学习路径

1. **循序渐进**
   - 按 Unite 顺序学习
   - 不要跳过基础内容

2. **多次复习**
   - 学完一遍后多次复习
   - 利用测试模式巩固

3. **结合游戏**
   - 学习后玩游戏巩固
   - 轻松记忆不枯燥

4. **阅读故事**
   - 学完 Unite 后读故事
   - 在情境中应用词汇

### 学习技巧

**记忆单词**：
- 注意词性和性别
- 联想记忆法
- 多听发音模仿

**准备测试**：
- 确保每个 Section 都复习过
- 先玩游戏热身
- 不急于求成

**阅读故事**：
- 先看法语原文理解大意
- 遇到不懂的查看中文翻译
- 多次阅读加深印象

## 🔧 常见问题

### 学习相关

**Q: 为什么有些 Unite 是锁定的？**
A: 需要通过测试赚取足够的星星才能解锁。先完成前面的 Unite 学习和测试。

**Q: 测试可以重复做吗？**
A: 可以！重复测试可以刷更高的分数，获得更多星星。

**Q: 如何获得宝石？**
A: 完成游戏挑战可以获得宝石。未来会增加更多获取方式。

**Q: 单词发音听不清怎么办？**
A: 可以重复点击音频图标多次收听。建议使用耳机获得更好的音质。

### 故事书相关

**Q: 故事书如何解锁？**
A: 每个 Unite 的第一本故事书在完成测试（≥60%）后自动解锁。其他故事书需要用宝石购买。

**Q: 为什么有些故事没有中文翻译？**
A: 部分进阶故事只提供法语原文，旨在提高纯法语阅读能力。

**Q: iPad 上如何使用双页模式？**
A: 将 iPad 横屏放置即可自动切换到双页模式。

### 技术相关

**Q: 数据会丢失吗？**
A: 所有学习数据保存在设备本地，不会丢失。但卸载应用会清除数据。

**Q: 可以在多个设备上同步吗？**
A: 当前版本不支持云同步。未来版本会考虑添加此功能。

**Q: 应用需要联网吗？**
A: 不需要！所有内容都在本地，完全可以离线使用。
