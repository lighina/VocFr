# Storybook Transcript Template
# 故事书文本内容模板
#
# 使用说明：
# 1. 复制此文件并重命名为 transcript.txt
# 2. 删除所有注释行（以 # 开头的行）
# 3. 填写每一页的法语和中文内容
# 4. 确保文件使用 UTF-8 编码保存
#
# 格式说明：
# - 每页以 "=== Page N ===" 标记开始（N为页码，从1开始）
# - 法语文本以 [FR] 开头
# - 中文翻译以 [ZH] 开头（可选，如不需要可留空或写 "-"）
# - 页与页之间用空行分隔
#
# 纯法语模式：
# - 如果只需要法语，[ZH] 行可以写：
#   [ZH]          （空）
#   [ZH] -        （占位符）
#   或直接省略 [ZH] 行
#
# 注意事项：
# - 法语文本应简单易懂，适合初学者
# - 使用该Unite的核心词汇（80%以上）
# - 每句5-10个单词为宜
# - 故事应有起承转合，贴近生活场景
